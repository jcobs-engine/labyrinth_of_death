## Installation ##

1. Download and extract ZIP or clone the repository to your home directory.

2. Run XFCE4-Terminal

3. ```shell
   user@PC:~> cd repo-dir/
   user@PC:~> bash game.sh
   ```

## Game ##

If the game is started, a window opens named CONTROLLER. The focus must always be on this window to be able to control. If you touch a monster (M), you lose two HP. If you touch other players, you lose 1 HP if you have more than the other player. You lose 4 HP if you have less. After the "fight", the two players are teleported to their spawn points on the other side of the field (A: 0, B: 1, C: 2, D: 3). The portal ( **ยง** ) will be replaced by the trophy ( **@** ) when there are only two players left. You get two HP if you touch the hashtag ( **#** ).
